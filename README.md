# Flawed dotfiles

Personal configs for Hyprland, fish, kitty, and more.

## Contents

| Config | What |
|---|---|
| `hypr/` | Hyprland keybinds, monitors, env, animations |
| `fish/` | Shell config & aliases |
| `kitty/` | Terminal theme |
| `foot/` | Terminal (Wayland) theme |
| `fuzzel/` | App launcher |
| `starship.toml` | Prompt theme |
| `btop/` | System monitor |
| `fastfetch/` | System info |
| `cava/` | Audio visualizer |
| `.bashrc` / `.bash_profile` | Bash fallback config |
| `.gitconfig` | Git config |

## Flawed Shell

Desktop shell for Hyprland: [github.com/Flawedexa/Flawed-Shell](https://github.com/Flawedexa/Flawed-Shell)
